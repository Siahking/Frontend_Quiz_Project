# Quiz Web App

A simple interactive **quiz application** built using **HTML, CSS, and Javascript**. Users answer multiple-choice questions and recieve instant feedback based on their selections.

The quiz randomizes both **question and anser options** so each attempt feels different.

---

## Features
 - Multiple choice quiz questions
 - Randomized question order
 - Randomized anser order
 - Instant answer validation
 - Visual feedback:
    - Green border for correct answers
    - Red border for incorrect answers
 - Score tally after submission
 - Celebration animation for a perfect score
 - Responsive design for mobile, tablet, desktop, and large displays
 - "Try Again" button to restart the quiz

---

## How to run
1. Download or clone the repository
```
git clone https://github.com/Siahking/Software-Development-quiz-assignment.git
```
2. Open the project folder
3. Open **quiz.html** in your browser
No installation required
